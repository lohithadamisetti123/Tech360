My C Projects

1. Mini Shell
A lightweight command-line interpreter implemented in C. Supports built-in commands, process creation with `fork()`, and command chaining using `|`. Demonstrates process control and string parsing.
2. Sudoku Solver
A backtracking-based solver for 9x9 Sudoku puzzles. Efficiently fills the grid by checking row, column, and box constraints. Reinforces recursion and constraint satisfaction concepts.
3. File Compression (Huffman)
Implementation of Huffman coding to compress text files. Uses frequency tables and binary trees to reduce file size without data loss. Good for understanding greedy algorithms and trees.
4. Student Record System
A console-based system for managing student records. Allows adding, deleting, updating, and viewing records stored in files. Demonstrates file I/O and struct usage in C.
5. Snake Game (Console)
A classic snake game using `ncurses` for UI. Features real-time movement, food generation, and score tracking. Showcases loops, conditionals, and terminal-based graphics.
6. Bank Management System
Simulates basic banking operations like deposit, withdrawal, and account creation. Uses file storage for account data. Emphasizes data structures and modular programming.
7. Calendar Generator
A CLI tool that prints formatted monthly calendars. Computes leap years and starting days accurately. Focuses on date logic and formatting output in C.
8. Tic Tac Toe
Console-based multiplayer Tic Tac Toe game. Includes turn-based logic and win/tie detection. Great for learning conditionals and array manipulation.
