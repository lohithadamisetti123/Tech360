My DSA Projects

1. Sorting Visualizer
An interactive visual tool that demonstrates how sorting algorithms work in real-time. Users can adjust array size and speed, and observe comparisons and swaps during execution of algorithms like Bubble Sort, Merge Sort, and Quick Sort. Great for learning and teaching core sorting logic.
2. Pathfinding Algorithms
A grid-based visual simulation of pathfinding techniques including A*, Dijkstraâ€™s algorithm, and Breadth-First Search. Allows wall placements, start/end adjustments, and shows how different algorithms explore the grid to find the shortest path.
3. Binary Tree Operations
Implementation of various binary tree functionalities in C++, such as insertion, deletion, in-order, pre-order, and post-order traversals. Includes console-based visual outputs and optional depth calculation. Helps understand tree structures and recursive algorithms.
4. LRU Cache
Java-based Least Recently Used (LRU) cache implementation using LinkedHashMap. Provides constant-time complexity for insertion and retrieval. Simulates memory cache behavior, often used in systems that need efficient eviction policies.
5. Graph Algorithms
C++ implementations of essential graph algorithms like Depth First Search, Breadth First Search, and Kruskal's Minimum Spanning Tree algorithm. Supports both directed and undirected graphs using adjacency lists, and includes weighted edge handling.
6. Stack and Queue Simulator
An animated interface demonstrating how stacks (LIFO) and queues (FIFO) operate. Supports push, pop, enqueue, dequeue actions with visual feedback. Ideal for understanding data structure behavior through intuitive simulations.
7. Dynamic Programming Playground
A collection of classical dynamic programming problems such as Knapsack, Longest Common Subsequence, and Matrix Chain Multiplication. Each problem includes bottom-up and top-down implementations with memoization and time complexity analysis.
8. Trie Data Structure
Java implementation of a Trie (prefix tree) with support for insert, search, delete, and autocomplete operations. Efficiently handles word storage and prefix matching, useful in applications like spell-checking and predictive text.
